# [learn git](https://github.com/tower-town/learn-git)

这是一个`git`的学习过程手册。一般的教程思路——讲解各个命令的作用。而随着现在涌现出一些新的`git`管理概念的出现，往往`git`未及时跟进,致使之后的学习者需要自己慢慢摸索。而这个`lear-git`将逐步跟进`git`的管理概念与`GitHub`的应用，介绍`git`及其涉及到的领域
